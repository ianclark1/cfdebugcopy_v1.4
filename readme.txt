1. Welcome
-----------------------------------------------------------------------

Welcome to CF Debug Copy, a Firefox extension that allows you to copy
selected ColdFusion queries from the debug output to the clipboard:

3. Installation
-----------------------------------------------------------------------

To install the CF Debug Copy extension, extract the .zip installation
package to a folder on your computer. Next go to the File/Open dialog
in Firefox and select the cfdebugcopy_vX.X.xpi, located in the extracted 
installation package, and open it. Be sure to restart Firefox when 
asked.
